# Relatório de Vendas — Sample Power BI

## Descrição do Projeto

Este projeto apresenta um Relatório de Vendas desenvolvido no Power BI, utilizando o dataset sample oficial, com foco na análise de desempenho comercial, custos, descontos e volume de vendas.

O relatório foi construído com ênfase em interatividade, usabilidade e experiência do usuário, incorporando elementos visuais que facilitam a navegação e a análise dos dados de forma intuitiva.

## Objetivo da Análise

O objetivo deste projeto é analisar o desempenho de vendas ao longo do tempo, fornecendo uma visão clara e interativa dos principais indicadores comerciais, com foco em:

• Análise do total de vendas

• Avaliação de unidades vendidas

• Análise de custos e descontos

• Evolução das vendas por mês

• Distribuição das vendas por segmento, produto e país

• Exploração dos dados por meio de interações dinâmicas

## Principais Insights

A análise demonstra um total de vendas de 118,73 milhões, com 1,13 milhão de unidades vendidas, evidenciando o volume expressivo de transações no período analisado.

Os descontos totalizam 9,21 milhões, impactando diretamente o resultado financeiro, enquanto os custos alcançam 101,83 milhões, reforçando a importância do controle de despesas para melhoria da rentabilidade.

Observa-se que determinados produtos e segmentos concentram a maior parcela das vendas, enquanto a análise geográfica permite identificar países com maior representatividade no faturamento.

A visualização temporal evidencia variações relevantes ao longo dos meses, permitindo identificar períodos de maior e menor desempenho comercial.

## Conclusão

O relatório fornece uma visão clara e interativa do desempenho de vendas, permitindo ao usuário explorar os dados de forma dinâmica e intuitiva.

A utilização de elementos interativos, como botões de navegação entre páginas e limpeza de filtros, demonstra a aplicação prática de recursos avançados do Power BI para melhorar a experiência do usuário e apoiar a tomada de decisão baseada em dados.

Este projeto reforça o domínio de Power BI, ETL, design de dashboards, sendo um excelente exemplo para portfólio profissional na área de Business Intelligence e Análise de Dados.
